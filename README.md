# RentInLondon PRO

Vista previa estática del proyecto RentInLondon PRO.

## Cómo usar

1. Clona el repositorio.
2. Abre `index.html` en tu navegador (doble clic o arrastrando el archivo).
3. Ajusta los filtros para explorar las propiedades de ejemplo.

No se requiere instalación de dependencias ni servidor adicional.
